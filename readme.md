Este proyecto contiene una propuesta de solución al ejercicio 10, el cual simula una competencia de cocina con 5 participantes, 3 jueces, de varias rondas, calculando ganadores parciales por rondas y una tabla de posiciones definitiva.

El proyecto se divide en algunos directorios:
- '\src': contiene el archivo 'competencia.py' con los datos de las rondas y el procesamiento de la competencia, que devuelve la acumulación de puntajes y los ganadores. 
- '\notebooks': contiene el cuaderno interactivo 'ejercicio10.ipynb', el cual importa la lógica desde 'src', para mostrar la ejecución y la tabla de resultados.

Para su ejecución, se debe tener instalado Python y Jupyter Notebook.

Pasos:
1) clonar este respositorio en la máquina local mediante Git. 
2) abrir la carpeta principal del proyecto en una terminal.
3) instalar la dependencia de Jupyter (para poder ejecutar el archivo '.ipynb')
